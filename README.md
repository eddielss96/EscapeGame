# 禁忌之門 — 密室逃脫 (The Forbidden Chambers)

這是一款採用純前端技術（HTML / CSS / JavaScript）打造的微恐主題單頁式網頁解謎遊戲。玩家將扮演調查員，在充滿詭異氛圍的宅邸中，解開一道道機關鎖，尋找逃脫的生路。

遊戲內建了強大的「後台關卡編輯器」，讓你可以直接在網頁上自訂題目、修改設定，並將關卡資料匯出/匯入分享給其他人。

---

## ✨ 核心特色

* **純前端架構：** 無需任何後端伺服器或資料庫，下載單一 HTML 檔案點擊即可遊玩。
* **沉浸式恐怖氛圍：** 結合滑鼠/觸控追蹤的「手電筒遮罩」視覺效果、雜訊背景，以及文字 Glitch（故障）特效。
* **動態音效引擎：** 使用內建的 Web Audio API 程式化生成所有背景音樂（BGM）與環境音效（SFX），完全不需額外載入外部音訊檔案。
* **豐富的機關鎖型：** 內建高達 10 種截然不同的互動解謎題型。
* **自動進度儲存：** 運用瀏覽器 `localStorage`，隨時記錄玩家通關進度、使用時間與提示次數，關閉網頁也不怕遺失。
* **內建管理後台：** 提供完整的視覺化關卡編輯器，支援拖曳排序、JSON 匯出/匯入、全域規則設定（倒數計時、提示上限等）。

---

## 🧩 支援的解謎題型

系統內建 10 種獨立的機關題型，可在後台自由切換與設定解答：

| 題型代碼 | 關卡名稱 | 互動方式說明 |
| :--- | :--- | :--- |
| `text` | 文字填空 | 基礎文字輸入，支援自訂佔位文字與解答（忽略大小寫）。 |
| `choice` | 選擇題 | 點擊多個選項按鈕進行選擇。 |
| `safelock` | 數字密碼鎖 | 透過上下按鈕轉動數字轉盤，支援自訂 2~8 位數。 |
| `gridlock` | 九宮格圖形鎖 | 依據正確順序點擊 3x3 陣列中的符文。 |
| `morse` | 摩斯密碼敲擊鎖 | 判斷長按（≧400ms）與短按（<400ms）輸入指定序列。 |
| `rotaryphone` | 古董撥號電話 | 模擬復古轉盤電話的輸入方式，撥打正確的分機號碼。 |
| `cryptex` | 同心符文圓盤 | 旋轉 2~5 層的密碼環，將目標符號對齊於中央指針。 |
| `compass` | 羅盤方位鎖 | 點擊八方位按鈕，輸入正確的移動路徑序列。 |
| `wirebox` | 斷路配電盤 | 將不同顏色的電線準確連接到對應的編號插槽中。 |
| `balance` | 罪惡天平 | 將自訂重量的物品分配至左右秤盤，達成兩端平衡或指定重量。 |

## 🚀 遊玩與安裝方式

1.  **下載檔案：** 將提供的程式碼儲存為 `index.html`（或任何自訂名稱的 HTML 檔案）。
2.  **開啟遊戲：** 直接使用任何現代瀏覽器（Chrome, Edge, Safari, Firefox）雙擊開啟該檔案。
3.  **開始挑戰：** 閱讀遊玩須知後，點擊「踏入禁忌之門」開始計時挑戰。
4.  **音效控制：** 遊戲左下角提供音量滑桿與 BGM/SFX 的靜音切換按鈕。

## ⚙️ 後台管理指南

遊戲內建一個隱藏的管理後台，方便遊戲主持人（GM）或創作者設計專屬的密室逃脫。

**如何進入後台：**
1.  在遊戲畫面的**右下角**，找到一個帶有齒輪圖示的「後台」按鈕。
2.  **連續快速點擊該按鈕 5 次**（點擊時會出現紅色計數徽章）。
3.  跳出系統驗證視窗後，輸入管理員密碼（預設為：`admin`）。
4.  按下 `Enter` 或點擊確認即可進入編輯模式。
5.  **快捷鍵：** 亦可使用鍵盤 `Ctrl` + `Shift` + `A` 直接喚出密碼輸入框。

**後台功能說明：**
* **全域設定：** 可調整最大提示次數、切換正計時/倒數模式、修改遊玩須知文本，以及更改後台登入密碼。
* **關卡管理：** 點擊「＋ 新增關卡」建立新題目，或在左側選單點選現有關卡進行編輯。
* **拖曳排序：** 鼠標按住左側關卡列表的 `≡` 圖示，上下拖曳即可快速改變關卡順序。
* **跳關測試：** 提供「直接跳關（繼續計時）」與「跳關並重設計時」按鈕，方便創作者測試特定機關。
* **資料備份：** 透過「📤 匯出 JSON」可將所有關卡下載至電腦；「📥 匯入 JSON」可載入他人的關卡檔（匯入會覆蓋現有進度）。

## 🛠 技術與相容性說明

* **無外部資源依賴：** 除了 Google Fonts（Creepster, Special Elite, Share Tech Mono）用於渲染恐怖字體外，不依賴任何外部圖片檔或外部音效檔。
* **RWD 響應式設計：** 介面適配桌上型電腦與行動裝置（手機/平板），觸控與滑鼠操作皆完全支援。
* **資料清除：** 若需完全重置遊戲與後台狀態，可於後台點擊「重載預設關卡」，或在瀏覽器開發者工具（F12）的 Application 標籤中清除 LocalStorage。
